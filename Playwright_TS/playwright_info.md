#Built-in Locators

1. page.getByRole() --> Locate elements by accessibility roles like button, checkbox, heading, etc.
2. page.getByText() --> Locate by visible text
3. page.getByLabel() --> Locate form controls using associated label text.
4. page.getByPlaceholder() --> Locate inputs via placeholder text.
5. page.getByAltText() --> Locate images by their alternative text( alt attribute).
6. page.getByTitle() --> Locate elements by their title attribute.
7. page.getByTestId() --> Locate by a custom data attribute like data-testid.
